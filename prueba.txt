Comandos Git 
git init: Crear repocitorio local
git config --global user.email "tu email" : Configurar usuario en Git (Mail de conexion en Git)
git config --global user.name "Nombre en Repo" : Configurar usuario en Git (Nombre Usuario)
git remote add origin [URL] : Crea conexion con Repositorio en GitHud
git push -u origin [Nombre de la Rama]: Sube datos a GitHud
git branch : Muestra la Rama
git clone [URL]: Clona Repositorio de Git en Computador
git checkout -b [Nombre Reama]: Crea Rama
